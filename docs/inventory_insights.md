# Inventory Insights

## Executive Recommendations

- Use forecast deltas to prioritize replenishment reviews before stockouts occur.
- Monitor promotional weeks separately because demand lift can distort baseline planning.
- Track forecast error by product and store to identify where model performance needs improvement.
- Convert forecast output into an action queue so planners see what to do next, not only what changed.

## KPI Layer

- Actual units sold
- Forecast units
- Forecast delta
- Mean absolute error
- Replenishment action count
- Category-level demand movement

# AI Retail Demand Forecasting & Inventory Optimization Platform

## Business Problem

Retailers often suffer from stockouts and excess inventory due to poor demand forecasting.

## Solution

This project builds an end-to-end AI-powered forecasting system that predicts product demand and generates inventory optimization recommendations.

## Tech Stack

- Python
- SQL
- PostgreSQL
- Power BI
- XGBoost
- Prophet
- AWS

## Project Architecture

Raw Data → ETL Pipeline → PostgreSQL Warehouse → Feature Engineering → Forecasting Model → Power BI Dashboard

## Key Features

- Demand Forecasting
- Inventory Optimization
- Automated Data Pipeline
- Executive Dashboard
- Sales Analytics

## Dataset

M5 Forecasting Dataset
